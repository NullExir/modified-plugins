# [L4D2] Grenadeluncher No Team Collision

让榴弹穿过生还者队友.

## 依赖

[Collision Hook](https://github.com/voided/CollisionHook)


## ConVar

```
// 开启插件.
l4d2_genade_launcher_no_collision_enable 1
```
